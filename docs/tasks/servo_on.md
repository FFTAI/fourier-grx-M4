# 执行器使能

## 任务信息

任务值 (TID): **35**

任务描述：

- 将所有执行器使能，电机启动输出力矩。

任务参数：

该任务无可配置参数。

## 模块信息

该任务暂无独立模块指令，不支持通过模块接口进行分关节控制。

## 接口信息

状态接口：

| 接口参数   | 接口映射关系                  | 说明                |
|--------|-------------------------|-------------------|
| 任务启动标志 | `task.flag_task_start`  | 使能过程中为 SET        |
| 任务结束标志 | `task.flag_task_finish` | 所有执行器使能完成后置为 SET  |

指令接口：

该任务无需额外指令参数，发送 TID=35 即可触发。
