# 执行器重启

## 任务信息

任务值 (TID): **41**

任务描述：

- 将所有执行器重启，重新加载配置和状态。
- 需要注意，重启后，所有执行器的状态和配置参数将被重置为默认值。

任务参数：

该任务无可配置参数。

## 模块信息

该任务暂无独立模块指令，不支持通过模块接口进行分关节控制。

## 接口信息

状态接口：

| 接口参数   | 接口映射关系                  | 说明                         |
|--------|-------------------------|----------------------------|
| 任务启动标志 | `task.flag_task_start`  | 重启过程中为 SET                 |
| 任务结束标志 | `task.flag_task_finish` | 所有执行器重启完成后置为 SET（约需数秒）     |

指令接口：

该任务无需额外指令参数，发送 TID=41 即可触发。重启完成后执行器将恢复为失能状态，需重新发送 Servo On（TID=35）使能。
