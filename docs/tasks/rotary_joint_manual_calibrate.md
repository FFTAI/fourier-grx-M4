# 手动校准（旋转关节）

## 任务信息

任务值 (TID): **4121**

任务描述：

- 用于用户将旋转关节**手动摆放至期望零位**后，触发软件层零点写入。
- 校准分两步依序执行：
  1. **关节下电**：执行 ServoOff，用户可自由移动关节至目标姿态后保持静止。
  2. **设置零点**（TID 4105）：采样约 1 秒（50 个控制周期）的关节位置读数并取平均值，将该平均值设为零点（应用层 0 rad 参考位置）；完成后自动将关节目标位置重置为新坐标系下的实测位置，防止下一个任务使能时出现目标跳变冲击。
- 与自动校准（TID 4120）的区别：

  | | 手动校准（TID 4121） | 自动校准（TID 4120） |
  |---|---|---|
  | 零位确定方式 | 用户手动摆放关节 | 机器自动驱动至机械限位后回退 |
  | 适用场景 | 快速现场校准、已知零位姿态 | 标准上电校准、初次使用 |
  | 需要悬挂 | 否（取决于目标零位姿态） | 是（需双脚离地） |

## 内部子任务说明

| 步骤 | TID  | 任务名称          | 说明                                                         |

## 注意事项与常见问题

### 执行前提条件

1. 所有执行器已 Servo On（TID=35）
2. 用户已确认旋转关节的期望零位姿态（任务触发后 ServoOff，再手动移动）

### 操作流程

1. 触发 TID 4121 → 执行器自动下电
2. 手动将旋转关节摆放至目标零位姿态，保持静止
3. 等待约 1 秒，程序自动完成采样并写入零点
4. `flag_task_finish` 置位后校准完成，任务自动切换回 IDLE

### 校准失败判断

- 校准成功：`flag_task_finish` 置位后，各旋转关节应用层位置应接近 0 rad（误差 < 0.1 rad）
- 若关节在采样期间有抖动，可重新触发 TID 4121 重新校准

### 使用限制

- 直线关节零点**每次上电必须**重新校准（TID=4210），不受本任务影响
- 本任务不可在运动任务执行过程中触发，请先退出当前任务再执行校准
