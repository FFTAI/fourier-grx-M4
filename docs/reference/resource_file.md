# 资源文件

* TOC
{:toc}

资源文件是 Fourier-GRX SDK 中用于存储和管理机器人的资源数据的文件，通常包括模型文件（机器人模型、神经网络模型）、参考数据、用户名密码信息等。

## 资源文件的路径

资源文件通常存储在 `~/fourier-grx/resource/` 目录下。不同机器人的资源文件存储在不同的子目录中。

例如，`FourierM4L` 机器人的资源文件存储在 `~/fourier-grx/resource/m4l/` 目录下。
