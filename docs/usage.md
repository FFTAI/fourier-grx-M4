## 🔧 校准与启动

### 关节零位校准

> ℹ️ **说明**：使用内置校准任务，可在【调试模式】下用手柄选中执行，也可在【开发者模式】下通过调用对应任务接口执行。

#### 启动前检查

**硬件**

- ✅ 机器人电源已接通
- ✅ 手柄正确连接
- ✅ 网络连接正常
- ✅ 插销已插入机器人关节定位孔（手动校准时）

**软件**

- ✅ SDK 已正确安装，`fourier-grx start` 可正常运行

#### M4L 上电校准顺序

> ⚠️ **M4L 系列机器人每次上电后必须按以下顺序执行校准，否则关节位置数据无效。**
>
> - 旋转关节（髋/膝）通过绝对值编码器保存零点，掉电不丢失
> - 直线关节（腿长）使用增量式编码器，**每次上电后必须重新校准**

| 步骤 | 任务名称 | TID | 说明 |
|------|----------|-----|------|
| 1 | 自动校准（直线关节） | 4210 | 腿长自动缩至最短后归零，约需 20s |
| 2 | 自动校准（旋转关节） | 4120 | 边界检测 → 下电垂落 → 零点写入，约需 60s |

#### 校准任务一览

| 机器人型号 | 任务 | TID | 说明 |
|------------|------|-----|------|
| M4L | 自动校准（直线关节） | 4210 | 每次上电必须执行 |
| M4L | 自动校准（旋转关节） | 4120 | 上电后按需执行 |
| M4L | 手动设置零点（直线关节） | 4203 | 手动插销对齐后使用 |
| M4L | 手动设置零点（旋转关节） | 4103 | 手动插销对齐后使用 |

#### 校准验证

校准完成后，终端会输出各关节的偏差数组：

- ✅ **成功**：所有值接近 0（偏差 < 1°）
- ❌ **失败**：出现大于 1 的值，需重新执行对应校准任务

按两次 `Ctrl+C` 退出程序。

#### 注意事项

**插销管理**

- 手动校准（TID 4103 / 4203）：执行前先插入插销
- 自动校准（TID 4120 / 4210）：无需插入插销
- 校准完成后，**必须拔出所有插销**再执行运动任务

**安全**

- TID 4210 执行时腿长机构自动运动，请保持安全距离
- TID 4120 执行时旋转关节缓慢运动至机械限位，确保关节无遮挡
- 紧急停止按钮保持可及

**故障处理**

- 自动校准超时：检查驱动器是否报警，执行 Clear Fault（TID 34）后重试
- 校准完成但运动异常：重新执行对应方向的校准任务
- 多次失败：联系技术支持并保存终端日志

### 程序开机自启动

```bash
fourier-grx enable_service   # 开启开机自启动
fourier-grx disable_service  # 关闭开机自启动
```

> ℹ️ 若机器人配置为手柄遥控模式，开启自启动后请确保手柄在开机前已连接并处于唤醒状态。

## 📋 日志与数据

### 启用/关闭程序运行日志记录功能

程序运行日志默认**开启**，记录位于 `~/fourier-grx/log/`。

如需关闭，修改启动脚本 `run.sh`：

> ⚠️ 关闭后将无法记录程序活动，可能影响后续调试排查。

```bash
# 原始（保留日志）
stdbuf -oL $FOURIER_GRX_HOME/run.bin --config=${config_file_path} \
| tee $FOURIER_GRX_HOME/log/${log_file_name}

# 修改后（丢弃日志）
stdbuf -oL $FOURIER_GRX_HOME/run.bin --config=${config_file_path} \
| tee /dev/null
```

### 启用/关闭机器人数据日志记录功能

机器人数据记录（关节位置/速度/力矩、IMU 等）默认**关闭**，启用步骤如下：

**第一步：创建记录专用配置文件**

复制当前配置文件并命名为 `config_M4L_T1_record.yaml`，在文件末尾添加：

```yaml
record:
  enable: true
  path: "~/fourier-grx/record/m4l"
```

**第二步：切换启动模式**

打开 `~/fourier-grx/run.sh`，将 `run_type` 改为 `record`：

```bash
run_type="record"
```

**第三步：启动**

```bash
fourier-grx start
```

> ℹ️ 恢复正常模式：将 `run_type` 改回原值，或通过 `fourier-grx config` 切换配置文件。

#### 数据格式说明

记录文件为逗号分隔的 `.log` 文件（UTF-8 编码），可用 Excel 或其他工具分析。

| 字段 | 含义 |
|------|------|
| `Timestamp` | 时间戳 |
| `imu_quat_{i}` | IMU 四元数分量 i |
| `imu_euler_{i}` | IMU 欧拉角分量 i |
| `imu_ang_vel_{i}` | IMU 角速度分量 i |
| `imu_lin_acc_{i}` | IMU 线加速度分量 i |
| `jm_pos_{i}` | 关节 i 测量位置 |
| `jm_vel_{i}` | 关节 i 测量速度 |
| `jm_tor_{i}` | 关节 i 测量力矩 |
| `jm_cur_{i}` | 关节 i 测量电流 |
| `jt_pos_{i}` | 关节 i 目标位置 |
| `jt_vel_{i}` | 关节 i 目标速度 |
| `jt_tor_{i}` | 关节 i 目标力矩 |

关节索引说明参见 [关节序列](/fourier-grx-M4/docs/reference/joint_sequence) 文档。
